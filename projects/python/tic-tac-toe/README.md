# Proyecto - Tres en Raya

¡Bienvenido/a al proyecto de Tres en Raya!

Este es un proyecto escrito en el lenguaje **Python**.

Consiste principalmente en el clásico juego de Tres en Raya.
Se crea un tablero de tres por tres.
Los dos jugadores deben intentar hacer tres en raya.
El primer jugador que lo consiga será el ganador.
El programa avisará cuando un jugador haya ganado o cuando haya empate.
