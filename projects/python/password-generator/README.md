# Proyecto - Generador de Contraseñas

¡Bienvenido/a al proyecto de Generador de Contraseñas!

Este es un proyecto escrito en el lenguaje **Python**.

Consiste principalmente en que el usuario debe elegir unas opciones para que el programa genere una contraseña segura.
El usuario deberá indicar si desea incluir letras.
El usuario deberá indicar si desea incluir números.
El usuario deberá indicar si desea incluir símbolos.
El usuario deberá introducir la longitud de la contraseña, entre 4 y 100 caracteres, ambos incluidos.
Una vez introducidos todos los parámetros, se generará una contraseña.
