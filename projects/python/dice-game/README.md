# Proyecto - Juego de Dados

¡Bienvenido/a al proyecto de Juego de Dados!

Este es un proyecto escrito en el lenguaje **Python**.

Consiste principalmente en que el usuario debe elegir uno o varios dados.
Puede elegir entre dados de 4, 6, 8, 10, 12 y 100 caras.
Una vez que el usuario haya elegido todos los dados que quiera, se lanzarán los dados.
El programa mostrará el resultado de los diferentes dados.
