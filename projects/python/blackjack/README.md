# Proyecto - Veintiuno

¡Bienvenido/a al proyecto de Veintiuno!

Este es un proyecto escrito en el lenguaje **Python**.

Consiste principalmente en que el usuario debe jugar al clásico juego de Blackjack.
El objetivo es acercarse lo más posible a 21 puntos sin pasarse.
El usuario y la computadora recibirán dos cartas cada uno.
El usuario podrá elegir si desea recibir más cartas o plantarse.
El programa calculará el puntaje total y comparará el resultado con el de la computadora.
Se le informará al usuario si ha ganado, perdido o empatado la partida.
