# Proyecto - Piedra Papel Tijera

¡Bienvenido/a al proyecto de Piedra Papel Tijera!

Este es un proyecto escrito en el lenguaje **Python**.

Consiste principalmente en que el usuario debe elegir entre piedra, papel o tijera.
Después, el programa generará una opción aleatoria.
Se compararán la opción del jugador y la del programa.
Se le dirá al usuario si ha empatado, ganado o perdido.
