# Proyecto - Adivina el Número

¡Bienvenido/a al proyecto de Adivina el Número!

Este es un proyecto escrito en el lenguaje **Python**.

Consiste principalmente en que el usuario debe intentar adivinar un número entre el 1 y el 100, ambos incluidos.
Si el usuario introduce un número más alto, se le avisará que el número es menor.
Si el usuario introduce un número más bajo, se le avisará que el número es mayor.
Una vez el usuario introduzca el número correcto, se le avisará y se le proporcionará el número total de intentos que ha realizado.
