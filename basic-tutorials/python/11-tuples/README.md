# Tutorial Básico - Tuplas

¡Bienvenido/a al tutorial básico de Tuplas!

Este es un tutorial básico escrito en el lenguaje **Python**.

Consiste principalmente en explicar el concepto tuplas, cómo se utiliza y por qué es fundamental en programación.
El concepto de tuplas es fundamental ya que permite almacenar múltiples elementos en una sola variable de manera inmutable, facilitando la agrupación de datos que no deben ser modificados durante la ejecución del programa.
