# Tutorial Básico - Listas

¡Bienvenido/a al tutorial básico de Listas!

Este es un tutorial básico escrito en el lenguaje **Python**.

Consiste principalmente en explicar el concepto listas, cómo se utiliza y por qué es fundamental en programación.
El concepto de listas es fundamental ya que permite almacenar múltiples elementos en una sola variable, facilitando la organización y manipulación de colecciones de datos en un programa.
