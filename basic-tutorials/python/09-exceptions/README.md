# Tutorial Básico - Excepciones

¡Bienvenido/a al tutorial básico de Excepciones!

Este es un tutorial básico escrito en el lenguaje **Python**.

Consiste principalmente en explicar el concepto excepciones, cómo se utiliza y por qué es fundamental en programación.
El concepto de excepciones es fundamental ya que permite gestionar errores que ocurren durante la ejecución del programa de manera controlada, asegurando que el programa pueda manejar situaciones imprevistas sin detenerse abruptamente.
