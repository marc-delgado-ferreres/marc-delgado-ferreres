# Tutorial Básico - Bibliotecas

¡Bienvenido/a al tutorial básico de Bibliotecas!

Este es un tutorial básico escrito en el lenguaje **Python**.

Consiste principalmente en explicar el concepto bibliotecas, cómo se utiliza y por qué es fundamental en programación.
El concepto de bibliotecas es fundamental ya que permite reutilizar código y aprovechar módulos preexistentes para realizar tareas específicas, lo que acelera el desarrollo y mejora la eficiencia del código.
