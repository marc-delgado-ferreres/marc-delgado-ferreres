# Tutorial Básico - Funciones

¡Bienvenido/a al tutorial básico de Funciones!

Este es un tutorial básico escrito en el lenguaje **Python**.

Consiste principalmente en explicar el concepto funciones, cómo se utiliza y por qué es fundamental en programación.
El concepto de funciones es fundamental ya que permite encapsular bloques de código en unidades reutilizables, facilitando la modularidad, la organización del código y la reducción de la redundancia en los programas.
