# Tutorial Básico - Diccionarios

¡Bienvenido/a al tutorial básico de Diccionarios!

Este es un tutorial básico escrito en el lenguaje **Python**.

Consiste principalmente en explicar el concepto diccionarios, cómo se utiliza y por qué es fundamental en programación.
El concepto de diccionarios es fundamental ya que permite almacenar pares de clave-valor, facilitando la organización y acceso rápido a los datos basados en claves únicas.
